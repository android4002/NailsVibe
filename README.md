# NailsVibe — сайт студии маникюра с панелью управления

Сайт для мастера маникюра. Проект включает в себя интерактивную публичную часть для клиентов и административную панель для управления контентом, пользователями и отзывами.

## Функционал сайта

### Публичная часть
* **Адаптивный интерфейс**: Разметка в виде Bento Grid (сетки блоков), которая автоматически перестраивается под мобильные устройства и ПК.
* **Интерактивный калькулятор**: Выбор нескольких услуг с автоматическим подсчетом суммарной стоимости и времени. Список выбранных услуг копируются в буфер обмена для последующего добавления в форму записи.
* **Карточки услуг**: При клике на услугу открывается модальное окно с описанием, ценой, длительностью, слайдером "До/После" (если загружены соответствующие фото) и галереей работ (поддерживает просмотр нескольких фото и видеороликов).
* **Портфолио**: Раздел выполненных работ с фильтрацией по категориям (маникюр, педикюр, дизайн и т.д.).
* **Атмосфера кабинета**: Видео-плеер для демонстрации кабинета и преимуществ студии.
* **Контакты и статус работы**: Интеграция с Яндекс.Картами. Статус работы (открыто/закрыто) рассчитывается автоматически по московскому времени на основе текущего часа.
* **Автономность (Offline-first)**: Библиотеки Alpine.js и плагины сохранены локально в папке `src/`. Это исключает задержки при загрузке внешних CDN и обеспечивает работоспособность скриптов.
* **Сохранение скролла**: Умное запоминание позиции прокрутки отдельно для главной страницы и страницы портфолио, адаптированное под мобильные браузеры (iOS Safari / BFCache).

### Панель управления (Админка)
* **Редактор блоков**: Возможность скрывать разделы на главной странице и менять их порядок отображения кнопками перетаскивания.
* **Редактор услуг и категорий**: Создание, изменение цен, длительности, описания и удаление услуг. Управление категориями прайса.
* **Загрузка медиафайлов**: Загрузка изображений и видео для услуг и портфолио напрямую на сервер через встроенный загрузчик.
* **Управление отзывами**: Список отзывов с фильтрацией по источникам (Яндекс, DIKIDI, добавленные вручную). Возможность скрывать выбранные отзывы с сайта.
* **Автоматический импорт отзывов**: Сервер раз в 3 часа опрашивает API виджета DIKIDI, импортирует новые отзывы, очищает дубликаты и сохраняет их в базу.
* **Черновик данных**: Автосохранение внесенных изменений в локальный черновик каждые 15 секунд для предотвращения потери данных при случайном закрытии вкладки.
* **Управление пользователями и ролями**: Создание учетных записей с ролями администратора и редактора. Права доступа разграничены на уровне сервера.
* **Резервное копирование**: Экспорт всей базы данных сайта в JSON-файл и восстановление из бэкапа через интерфейс админки.

## Демонстрационный доступ в админку
При локальном запуске проекта войти в панель управления можно по адресу: `http://localhost:3005/admin.html`
* **Логин**: `admin`
* **Пароль**: `nailsvibe2026`

*(При работе в демонстрационном режиме локальные изменения применяются только на запущенном экземпляре сервера и не влияют на реальный рабочий сайт).*

## Стек технологий
* **Фронтенд**: HTML, Tailwind CSS, Alpine.js (с плагином Collapse).
* **Бэкенд**: Node.js, Express (API-сервер, авторизация, загрузка файлов).
* **База данных**: Локальные JSON-файлы (`data/site_data.json` и `data/users.json`).
* **Безопасность**: JWT-токены в HttpOnly-куках, хеширование паролей bcryptjs, CORS-валидация.
